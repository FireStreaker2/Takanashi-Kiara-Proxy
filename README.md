## Setup via Terminal
```
$ git clone https://github.com/FireStreaker2/Takanashi-Kiara-Proxy
$ cd Takanashi-Kiara-Proxy
```
> Skip the first two steps if you deployed via one of the buttons above
```
$ npm i
$ npm start
```
